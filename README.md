# Rocket League Quick Chat Tool

A lightweight Windows/desktop macro tool for Rocket League that lets you bind
custom chat phrases to the `V`, `R`, and `F` keys. Press a trigger key while
scanning is enabled and the tool opens team or all chat, types your phrase,
and hits Enter for you.

## Features

- **Custom phrases per key** — set independent messages for `V`, `R`, and `F`.
- **Team or All chat** — choose the chat channel per phrase.
- **Global hotkeys** — `F6` enables scanning, `F7` disables it, from anywhere.
- **Key suppression (Windows)** — trigger keys are swallowed system-wide so
  they never reach the game itself, only the chat macro fires.
- **Safe against self-triggering** — synthetic keystrokes generated by the
  tool are ignored by its own listener, and it pauses while the tool window
  is focused (so you can type phrases without accidentally firing them).
- **Cooldown protection** — a short cooldown prevents accidental spam/rapid
  re-fires.
- **Persistent settings** — phrases and chat-channel choices are saved to
  `quickchat_config.json` next to the script and reloaded automatically the
  next time you launch the tool.
- **Dark-themed GUI** — built with Tkinter, rounded panels, live activity log.

## Requirements

- Python 3.9+
- [`pynput`](https://pypi.org/project/pynput/)

```bash
pip install pynput
```

Tkinter ships with most standard Python installs. On Linux, you may need to
install it separately (e.g. `sudo apt install python3-tk`).

> **Note:** Key suppression (so `v`/`r`/`f` don't reach the game) only works
> on Windows, since it relies on a `win32_event_filter` low-level keyboard
> hook. On other platforms the tool still works, but the trigger keys will
> also be sent to whatever window has focus.

## Usage

1. Run the script:
   ```bash
   python Custom_Quick_Chat_V3.py
   ```
2. Enter a phrase for `V`, `R`, and/or `F`, and pick **Team** or **All** chat
   for each.
3. Click **Start Listening**.
4. In-game, press **F6** to enable scanning and **F7** to disable it.
5. Press `V`, `R`, or `F` to fire the matching phrase.
6. Click **Stop & Exit** (or close the window) to quit — your phrases are
   saved automatically.

Leave a leading space in your phrases if you want a visual gap after the
chat-channel prefix Rocket League inserts.

## Configuration file

Settings persist in `quickchat_config.json`, created automatically in the
same folder as the script (or next to the `.exe` if bundled with
PyInstaller):

```json
{
  "phrases": { "v": "", "r": "", "f": "" },
  "chat_keys": { "v": "Team", "r": "Team", "f": "Team" }
}
```

Delete this file to reset to defaults.

## Disclaimer

This tool automates keystrokes for chat messages only — it does not read
game memory, alter gameplay, or provide any competitive advantage. Automating
inputs may still be against the rules of some platforms or tournaments; use
at your own discretion and risk.
